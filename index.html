Squi Docs

Handy-dandy documentation for using Squi (now that you've installed it). You can disable this by commenting out the Route::get() blocks in bundles/squi/start.php. Leaving it in should not affect your app's performance, but I support that flavour of paranoia :)

A really cool thing about these docs is that all of the examples are pretty much copy-pasta from the unit tests. Pretty cool, huh?

Lastly, these docs are in the github repo, so if you come across any deficiencies please help us all out by sending me a pull request.
